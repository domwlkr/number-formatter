Number Formatter
=========

A small library that adds commas to numbers

## Installation

    `npm install @kapeboi/number-fromatter`

## Usage

        var numFormatter = require('@kapeboi/number-formatter');

        var fromattedNum = numFormatter(35666);

    Output should be `35,666`
